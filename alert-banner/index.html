Role="alert" demo and notes

Things to know about role="alert"

  • The alert role is a live region. In essence it is the same as setting aria-live="assertive" and aria-atomic="true".
  • Content needs to be loaded dynamically to trigger an announcement to the screen reader user. (Some screen readers will announce content on load, but this behavior is not consistent across screen readers.) 
    
            <script>
              message = () => {
                document.getElementById("alert-me").textContent = "This may not be the log-in page you expected.";
              };
              message();
            </script>
  • In the static HTML there should be a placeholder div with role="alert". This preps the screen reader software to monitor that container for changes. 
    
              <div id="alert-me" role="alert">
              <!-- empty div is placeholder for dynamically entered content -->
              </div>
