Security recommendations

Don't use UW NetID passwords on other accounts

If you use the same password for different purposes, a cyberthief can learn the password from a less secure site or application and then use it to exploit other accounts. So never reuse your UW NetID password on any other account, and avoid re-using passwords across accounts in general.

Use and update antivirus

Anti-virus software can help protect computers and devices from various forms of infection, but it must be kept up to date. Members of the UW community can install a free version of Sophos anti-virus for home use and for unmanaged University-owned computers and devices.

Avoid using software downloaded from unknown websites

Avoid using software downloaded from unknown websites or peer-to-peer file-sharing services as they may include spyware.
